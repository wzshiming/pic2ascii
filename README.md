# pic2ascii

## Requirements

Go version >= 1.5.

## Download and install

``` shell
go get -u -v github.com/wzshiming/pic2ascii/cmd/pic2ascii
```

## Commands

``` shell
pic2ascii -c "MMWNXK0Okxou=:\"'.  " -f https://avatars0.githubusercontent.com/u/6565744 -w 90 -h 40
```
